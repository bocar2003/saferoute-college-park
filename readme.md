# SafeRoute College Park

SafeRoute College Park is a full stack web application
that helps UMD students understand crime trends
around College Park using maps and charts.

## Technologies Used

- Node.js
- Express.js
- Supabase
- Leaflet.js
- Chart.js
- Vercel

## Target Browsers

- Chrome
- Safari
- Firefox
- Edge
- iOS Safari
- Android Chrome
